About Pokedex

Pokedex application is a JavaScript app built with HTML, CSS, JavaScript and jQuery, which uses an external API.

Key Features

  • Load and display Pokemon data from an external source (API).
  • On user action (e.g., by clicking on a Pokemon on a list ),display details for that Pokemon in a modal.
  • Find a Pokemon on external API by a Pokemon's name and display in a modal.
  • User friendly error message.

Technologies used

  • HTML 5
  • CSS 3
  • Bootstrap 4
  • JavaScript
  • jQuery
